#### Redis 基本概念

- 名称： Remote Dictionary Server

redis 是速度非常快的非关系型(NoSql)内存键值数据库，可以存储键和五种数据类型。

键的类型只能字符串，值支持的数据类型有五种：字符串/列表/集合/有序集合/散列表

redis 支持很多特性，如可将数据持久化到硬盘中，使用复制来扩展读性能，使用分片来扩展写性能
