## script

Support Windows platform

- view help

  ```
  run -h
  ```

  -- merge
  "Merge files in doc directory"

  -- split
  "Split readme file"

  -- compile xxx
  "Compile java code"

  -- start xxx
  "Run java code"

- Merge files in doc directory

  ```
  run merge
  ```

- Split readme file

  ```
  run split
  ```

- Compile java code

  ```
  run compile xxx(java file name,do not need suffix '.java')
  ```

- Run java code

  ```
  run start xxx(class file name,do not need suffix '.class')
  ```
