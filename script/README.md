## script

Support Windows platform

- view help

```
// /Interview
run -h
```

-- merge
"Merge files in doc directory"
-- split
"Split readme file"
-- compile xxx
"Compile java code"
-- start xxx
"Run java code"

- Merge files in doc directory

```
// /Interview
run merge
```

- Split readme file

```
// /Interview
run split
```

- Compile java code

```
// /Interview
run compile xxx(java file name,do not need suffix '.java')
```

- Run java code

```
// /Interview
run start xxx(class file name,do not need suffix '.class')
```
